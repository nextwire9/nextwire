# NEXTWIRE One-Page Website

ملفات صفحة NEXTWIRE الجاهزة للرفع.

## الملفات
- `index.html`
- `style.css`
- `script.js`
- `assets/`

## تعديل الروابط لاحقًا
كل روابط المنصات موجودة داخل ملف `script.js` في مصفوفة `platforms`.

حاليًا الروابط موضوعة كـ `#` إلى أن يتم تزويد الروابط الرسمية.
